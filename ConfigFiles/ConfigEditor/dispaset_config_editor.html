Configure Zones/Countries

Configure the zones/countries to be included in the simulation:

  • Enter a unique code for each zone/country (e.g., BE, FR, DE)
  • Set each zone as Active (included in simulation) or Inactive
  • For active zones only, set MTS status (included in mid-term scheduling)
  • Use the + button to add more zones, or the trash icon to remove a zone

Active zones will be included in the "zones" list. Active zones with MTS status will also be included in the "mts_zones" list.
